### O projektu

Jednoduchá aplikace pro vyhledávání uživatelů na Githubu. Využívá Github API a moderní technologie jako React Hooks, TanStak Query, Axios, Tailwind a další. Data jsou cachována a aplikace je plně responzivní. Projekt je vytvořen jako součást [kurzu](https://www.udemy.com/course/modern-react-front-to-back/) na Udemy. Cílem bylo naučit se nové technologie a vyzkoušet si je v praxi.

### Použité technologie

- Astro
- Preact
- Tailwind
- TypeScript
- MDX
- GSAP

#### Zajímavé části kódu

```javascript
const useProfileSearch = () => {
  const context = useContext(UsernameContext);

  if (!context) {
    throw new Error("useProfileSearch must be used within a UsernameProvider");
  }

  const { username, setUsername } = context;

  const { error, data, refetch } = useQuery(
    ["repoData", username],
    () =>
      fetch(`${import.meta.env.VITE_GITHUB_API_USER}/users/${username}`).then(
        (res) => {
          if (res.ok) {
            return res.json();
          } else if (res.status === 404) {
            return null;
          } else {
            throw new Error("An error has occurred");
          }
        }
      ),
    {
      enabled: false,
    }
  );
```
