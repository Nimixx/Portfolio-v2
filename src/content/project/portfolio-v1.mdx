### O projektu

Moje první verze portfolia byla napsaná především tak aby ukazovala moje prpjekty a technologie se kterými pracuji. Snažil jsem se i o to aby byla flexibilni s ohledem na budouci přidávání projektů. Nicméně po nějakém čase jsem se rozhodnul přistupovat k celé stránce ještě jinak a tak začala vznikat celá nová verze, na kterou právě koukáte.

### Design

Vzhled byl vytvořen mnou a jeho cílem byla především jednoduchost, dostupnost prvků a stejně tak i responzivita celé aplikace.

### Použité technologie

- Vite
- React
- TypeScript
- Tailwind CSS
- Lucide icon

#### Ukázky kódu

```tsx
export default function IconStuck() {
	return (
		<section className="flex w-full flex-row flex-wrap items-start gap-2 pt-5 2xl:justify-between">
			{skills.map((skill, index) => (
				<TooltipProvider key={index}>
					<Tooltip>
						<TooltipTrigger asChild>
							<div className="h-10 w-10">
								<Icon icon={skill.icon} width="100%" height="100%" />
							</div>
						</TooltipTrigger>
						<TooltipContent>
							<p>{skill.tooltip}</p>
						</TooltipContent>
					</Tooltip>
				</TooltipProvider>
			))}
		</section>
	)
}
```
